↓

Send Verification Email

# 💻 Installation

Clone Repository

```bash
git clone https://github.com/simran-123-dev/Advanced-Auth-Security-Lab.git
```

Install Dependencies

```bash
cd client
npm install

cd ../server
npm install
```

Run Backend

```bash
npm run dev
```

Run Frontend

```bash
npm run dev
```
